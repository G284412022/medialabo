「Webコンテンツ作成の結果」

本テーマの学習目的

HTML,CSS,JavaScriptの三つに大きく分類されるものを学び,概要,構造と関係性を理解し、Webページの構造、見た目を細かく設定することにより簡易的なページを自分で作れるようになる。

プロジェクト課題について

番組表、グルメ情報、世界各地の天気の三つ中から私はプロジェクト課題に世界の天気を選んだ、世界の天気をサーバーからデータ取得して、表示するものを作成しました。 首都を自分で選択し検索することでその都市の天気、最低気温、最高気温、湿度、緯度、経度、風速、風向き、が表示されるようになっています。

    今回はDOM（Document Object Model）という Webコンテンツを表現するデータ構造について学んだ。DOMはWeb コンテンツの内容を読み取ったり，コンテンツに変更を加えたりする処理が書けるようになり、 DOM と JavaScript を組み合わせることによって，動きのある Webページを実現できるようになった。また、演習では写真表の追加、箇条書きの追加や削除を行いました。

    感想

    今回私はWebコンテンツ作成について学んだ。6回の授業の中でさまざまなことについて学んだ。6回の授業の中で私は2回目のWebページの装飾が一番面白いと感じた、 なぜなら自分で色を変えたり影をつけ、レイアウトを変えて変更した部分がページを更新した時に目に見えて変わったのがわかりやすく、自分で作っているという実感が得られとても満足した、 今まで受けた授業では動作の確認しかしなかったので、エラーなく動いたくらいにしか思わなかったが、自分が変えたところが実際に変わっているのをみると、とても感動したそしてとてもやりがいを感じた。 私が今まで使ってきたウェブページを作るのがこんなに大変で、手間のかかるものだとは思わなかった。今回の私は講義の内容は私自身初めて体験する内容になっていた。そのためなかなか理解が追いつかずなかなか思い通り進めることができなかった、 多くの課題をその日のうちに終わらせることができず、家に帰ってから進めることがほとんどであった、なので友達に質問することがとても多く、いつも助けてもらっていた。その中でも特に第4回のdomについての講義が一番難しかった、 domの講義では新しく覚える構文がたくさんあり、何をどの時に、どの順番で使うかなどとてもそれらを覚えるのに苦労した。他にも作成したページのレイアウトを変えようとしたのだなかなか上手くいかなかったり、変更したところが上手く表示されなかったりした。 今回作成したWEBページはそんな私がたくさんの友達から意見を聞きながらできるだけ簡略的に作ったものだ。なので要素の種類もできるだけ少なくしてわかりやすく、汎用性のあるものを探しながら使い工夫をした。 代わりににcssでの装飾で他の人より個性を出せるように努力をした、しかし限られた時間の中、他の科目の課題もあったため最初に自分が思い描いていたほどの装飾ができなかったのが悔しいと感じた。授業の中で過去に先輩が作成したページの中に、 背景が写真を使っているものがあり、私も挑戦したかったが時間が足りず叶わなかった。今回の講義で自分でWEBページを作るという貴重な体験ができたのでとても楽しかった、なのでもっと自分で調べて新しくWEBページを作ってみたいと思う。
